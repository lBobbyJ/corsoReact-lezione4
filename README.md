# React + Vite

Corso React | Lezione 4
Obiettivo della lezione

Creare una mini dashboard di gestione prodotti/inventario con:

- Inserimento nuovi prodotti
- Visualizzazione in Card riutilizzabili
- Filtri avanzati: categoria, prezzo
- Multi-sort: nome A–Z / Z–A, prezzo crescente/decrescente
- Ottimizzazione render con "useMemo" e "useCallback"
- Statelesse children + stato globale nel parent
- Persistenza in "localStorage"

Nota: ogni componente sarà indipendente, progetto separato dalle lezioni precedenti.



Tecnologie

- React + Vite
- Node.js
- Git
